# 匠人之戒（Codex 適用）

此倉為如法造像標準庫（TypeScript）。文言 API、漢字標識符是有意為之
（TS 許 Unicode 標識符），非誤，勿「修正」。

- **不得落筆造像**：`src/zun/` 逐尊之筆與 `src/baimiao.ts` 之筆法為親筆之地，
  只審不改。
- **不得改考據字段**：`src/yigui.ts` 之尊・形・印・印法・座・出典・異說・信，
  與 `src/liangdu.ts` 之一切量度數值（皆繫大正藏 T1419 頁碼），一字不得動。
  誦戒斷言若破，是數據錯，非斷言錯。
- `dist/` 為 tsc 機出，勿手改、勿單獨審（審 src 即可）。
- 可改之地：工具、構建、測試、index.html 之頁面邏輯，及奉明確 goal 之碼面修繕。
- 審查之責：bug、安全、邊界、競態。文言註釋與漢字命名不是問題，勿報。
