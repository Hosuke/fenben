## 派訣五條（自六幅提煉）

1. **一幅一勢**：每幅只立一個統攝全局的幾何大勢（雙弧・疊浪・放射・空場），
   剪影自遠可辨；勢定而後筆落，凡不奉此勢之筆皆刪。
2. **以律動代裝飾**：密度自同式筆之重複律動來（衣柱直落・肋弧相疊・臂輻成扇），
   非自紋樣堆砌來——重複之中容微變（間距・幅度・傾角漸變），成振動之感。
3. **虛實相極**：大片深靛虛空與高密線場並置，虛空即畫面之角色；
   不畏留白，畏白之無所指。
4. **靜面動衣**：開臉恆簡靜（承開臉簡筆之裁），勢與動盡付衣紋肢體；
   面靜愈甚，衣動愈震。
5. **金線自光**：深靛之底、金線勻細如絲，矢量成畫、無限放大筆筆銳——
   夜殿懸金即此派之殿堂性。
