# 風格館分層之制（2026-07-12 主人裁：尊身親筆、框飾遣匠、修蔽歸主）

十四席候展稿初驗盡墨（框飾密而尊身塌），遂改制：**一席三層，兩坊分職，主坊終審**。

## 三層

| 層 | 內容 | 誰筆 | 檔 |
|---|---|---|---|
| 一・景 | 背幕・山水・樹雲・焰光背光・龕陣空框——凡居尊身之後者 | Codex | `匠筆-codex/風格館/<席>-框飾層.js` export `景(ctx, 助)` |
| 二・尊 | 主尊與諸小尊之身——開臉・軀幹・手足・衣紋・持物 | **主坊親筆** | `匠筆-主坊/風格館/<席>-尊身層.js` export `尊(ctx, 助)` |
| 三・飾 | 邊框紋帶・座前供物・幢幡垂瓔——凡居尊身之前者 | Codex | 同框飾層檔 export `飾(ctx, 助)` |
| 終・修 | 修蔽（遮蔽次序之失）・正透視（榻座几案之線）・合成 | **主坊** | 尊身層檔可 export `修(ctx, 助)`（末筆） |

- 皆為瀏覽器 ES module（`.js`，倉內相對路徑 http 載入），畫於同一透明布，**繪序即蔽序**：景→尊→飾→修；末合成深底（蔽法之律）。
- 合成器：`匠筆-主坊/風格館/合席.cjs <席名>`（主坊具）→ `圖錄/風格館/<席>-候展稿.png`。
- Codex 於框飾層檔中 `export const 尊位 = {…}`——數值由主坊定（見下），Codex 原樣照錄不得擅改；景飾諸物繞此位佈局。

## 戒

- **尊自蔽其形**：尊身層（主尊與小尊皆然）開筆必先以 destination-out 清出自身
  silhouette（頭光盤・頭頸・軀臂・盤股諸形），景層背幕織紋不得穿面穿身——
  繪序即蔽序之推論，2026-07-12 齊烏崗巴首合成立此律。
- Codex 不畫任何身形（主尊・小尊・侍者・童子皆主坊）；龕陣留空框。
- 密度須自結構交疊，禁同印複拓（焰雨・斑點雨・同臉龕陣之病，初驗已列）。
- 兩坊皆：headless 親見乃報；不動 博物館.html・src/zun・baimiao・yigui・liangdu・dist；不 commit（主人另行明令）。
- 主坊尊身之判準仍=尼瓦爾-精密二稿之骨面手；每席並排自評而後合成。
- **チ鏡向律**（2026-07-13 吳家樣碼審所斬立律）：舉掌外向者（施無畏・說法），
  拇在橈側＝**朝中軸**；垂臂掌外向者（與願，解剖正位全旋後），拇在橈側＝**朝外側**。
  每掌必補拇側斷言；捻拇之梢回勾向掌，斷言繫拇**根節 p1** 毋繫梢。
- **不抹之律**（2026-07-13 館主親立）：凡完整之嘗試皆值一記，不必抹去——舊稿退役
  不刪；tools/存檔廊.py 自倉史掘各席歷代全稿出縮圖入 存檔.html（各帶 ID＝席名＋卷碼），
  館主指某幅之 ID 即可 `git show` 溯其全圖為參。每有退役重生，重跑存檔廊補錄。
- **斷言禁自比律**（2026-07-13 欽孜杖・鎌倉棒・宋白描柄・嘎孜弦四犯而立）：
  チ斷言所量必為實繪曲線之端點變量；常數與其定義式相比（恆零恆真）即自比，無效。
  軸線類（杖棒柄弦）取二軌實端之中為軸乃量；驗具（框飾驗）改後**必親復跑至綠**——
  匠人自報之綠不可信（斯瓦特・鎌倉二驗具皆曾自報綠而實敗）。
