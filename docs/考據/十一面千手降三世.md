# 考據 · 十一面觀自在・千手千眼・降三世明王（複雜像三尊，為多面多臂忿怒之制備）

> 2026-07-09 立卷（opus 代理五路）。〔待核〕〔闕〕如標；詳表全錄，此乃面塔臂扇之制之數據底。

## 十一面觀自在

- **根本原典**（玄奘譯《十一面神咒心經》T1071，CBETA 逐字）：「左手執紅蓮花軍持，
  展右臂以掛數珠及作施無畏手。其像作十一面，當前三面作慈悲相，左邊三面作瞋怒相，
  右邊三面作白牙上出相，當後一面作暴惡大笑相，頂上一面作佛面像。諸頭冠中皆作佛身。」
- 面塔之表：前三慈悲・尊左三瞋怒・尊右三白牙上出・後一大笑・頂一佛面（含本面共十一）。
  ※世界大百科左右逆記乃觀者向之述，從原典尊自身左右。
- 臂：二臂（玄奘系，日本通例）＝左紅蓮軍持・右垂數珠施無畏；四臂本不空譯，
  逐手持物〔待核〕。三昧耶形＝水瓶開蓮。
- 坐立：**獨尊＝立像**（聖林寺渡岸寺諸國寶皆立）；蘇悉地院曼荼羅像坐立臂數〔待核〕。
- 搩度判：善相菩薩十搩。

## 降三世明王（金剛薩埵之忿怒身，五大明王東方）

- **面臂**：四面八臂（東洋画題綜覽・デジタル大辞泉・東寺立像定說；三面八臂並錄辭書通例）；
  四面各三目；第四面在後常隱。
- **降三世印**（本手，≥4源）：兩腕胸前交叉、餘指金剛拳、**左右小指互相鉤結、
  左右頭指並豎**，印當心右。「檀慧相鉤竪進力」之漢文〔待核〕。
- **逐手持物三型並陳**：辭書型 右＝箭劍三鈷鈴／左＝索弓三鈷戟；畫題綜覽（金剛界畫像）
  右＝三鈷杵箭劍／左＝三鈷戟弓索；東寺立像 右＝金剛杵矢刀／左＝金剛戟弓索。
  右一手鈴／杵二說〔待核〕。
- **踏姿定說**（五源，東洋画題綜覽逐字複核）：「左足大自在天の頂を按じ、右足大自在天妃の
  乳所の上を踏む」——**左足按摩醯首羅頭頂、右足踏烏摩妃胸上**；立像（住降伏立相）。
  被踏者：シヴァ合掌、妃以手觸明王足；仰臥之判〔待核〕。展左展右逐字〔闕〕（東寺常展右）。
- **忿怒相**：三目・利牙出現・焰髮逆立・顰眉猛視・**「以蓮華面微笑而怒」**（理趣釋——
  微笑而怒之矛盾相，簡忿開臉之魂）；面色 正青右黃左綠後紅；**蛇飾臂釧不採**
  （屬軍荼利，此尊無典）。
- 胎藏持明院同名尊（伐折囉吽迦囉金剛）專屬像容〔待核〕——暫依金剛界型注待核勿臆造。
- 搩度判：忿怒八搩（T1419 之律）。
