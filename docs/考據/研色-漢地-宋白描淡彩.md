## 裁定之由

- **濃墨骨線**（雙源可核）：雙館方記錄逐字核實:故宮《維摩演教圖》頁「紙本,墨筆」(文物號新00146319,尺寸34.6×207.5cm皆符)+文化遺産オンライン《五馬図巻》「紙本墨画淡彩」(重要美術品・東博皆符)。RGB(46,40,35)暖調近黑,合碳墨於老紙之面貌(純碳本中性,微暖為老化攝影之果,原檔自稱「近似」不誤)。松煙/油煙之分屬通行製墨常識,無單件儀器實證,不礙大判。
- **淡墨積染**（雙源可核）：板倉聖哲之判兩刊獨立可核:日經2019「線の強弱だけで遠近感を示し、繊細な濃淡で人や馬の肌の質感と立体感を表す」經檢索復現原句;澎湃2019訪談「畫面色彩時與水墨塊面、線條重合,生成複雜的色調變化」逐字核實。館方「墨画淡彩」為旁證。惟兩源同出板倉一人之口,係同一鑑定意見之兩刊,非全然獨立之二鑑。RGB(128,116,104)中灰偏暖合稀墨罩老紙。
- **淡赭紅褐**（單源）：「先用墨線畫過之後,再上彩色,之後又上墨色」「馬身基本上以紅褐色及黃褐色二色漸層上彩,並且分為頭、胸、腹部數個部位區塊涂色」皆於澎湃2019板倉訪談逐字核實——然此為唯一詳述之源。「赭石(赤鐵礦)」係目鑑色相之推定:訪談原文僅言「紅褐色」不名顏料,五馬圖無公開XRF/拉曼(原檔doubts已自陳)。RGB(172,118,82)與稀赭罩染色相相符。
- **赭黃黃褐**（單源）：黃褐漸層之據同出澎湃2019板倉訪談一源(逐字核實「紅褐色及黃褐色二色漸層上彩」)。「土黃/褐鐵礦系」為顏料推定,無材料實證。RGB(186,148,94)合土黃罩染之色相,與淡赭紅褐拉開恰當之黃相差。
- **栗色鬚線**（單源）：「栗色的絡腮胡和唇上的鬍鬚質感是以彩色的線條與濃淡不一的墨線交相重疊來表現」於澎湃2019板倉訪談逐字核實,惟此一源。「赭石調墨」為配方推想,訪談僅言「栗色」。RGB(106,70,48)栗褐相符。以線施彩非染面之辨,亦僅此訪談所出。
- **淺朱**（單源）：2018澎湃報導「仅在缰绳和人面部施有浅朱砂色,且技法几乎仅为简单的平涂」「红色颇为单薄飘忽,呈透明状」逐字核實——然該文自注此語轉引「一些艺术类微信公号」之目鑑,屬二手;板倉2019證人面確有上彩(且修正為多層非平塗)而未名朱砂、未及韁繩。朱砂/鉛丹/朱膘無儀器判別:若實係鉛丹則老化變色當另計,原檔doubts自陳此漏。RGB(190,96,76)之單薄透明觀感與報導相符,薄施朱砂可解。
- **絹本古色**（雙源可核）：疑辨官親以同法重採本地檔獨立復現:Met 1980.276維摩不二圖亮帶中值(189,152,114)對所稱(188,152,114)幾全同;孝經圖scene02亮段(208,187,150)合所稱上界(212,191,154),scene06/10深段(123–128,97–102,69–73)近所稱下界(135,108,77)。兩件獨立絹本文物(Met 1980.276+Met 39895)老化色調交叉一致;Met API核實39895 medium逐字「Handscroll; ink and color on silk」。「非顏料・老化之果」之自我標注誠實。
- **紙本古色**（單源）：重下載dpm預覽圖s58def92950e24.jpg(實測1258×500px)復測得(182,162,138),與所稱(182,163,139)全同——採樣誠實可復現。然RGB之源僅此一幅白平衡不明之官網小圖,對實物紙色之保真不可考(原檔doubts已自陳)。澄心堂紙之文獻鏈反屬雙源俱核:湯垕《畫鑒》「不為著色獨用澄心紙為之惟臨摹古畫用絹素著色」維基文庫逐字核實+板倉訪談「前四組應該是澄心堂紙」;「原色近白、今為老化」之辨亦確。惟色值本身單源,從嚴判single-source。

## 存疑待核

- 花青查無實證：五馬圖所記諸色皆赭朱暖系（紅褐・黃褐・栗・淺朱），孝經圖CC0高清色相群集掃描180–270°青藍域零群集——宋白描淡彩施花青之說不錄，如需青色當另尋南宋淡設色卷軸實證
- 五馬圖無公開之顏料儀器分析（XRF/拉曼）——赭石・朱砂之判由目鑑色相與通行技法詞，非材料實證
- 孝經圖之color館方僅記medium未逐處說明；Richard Barnhart《Li Kung-lin's Classic of Filial Piety》(Met, 1993)章節頁碼未親核（Met頁面429未達）
- 澄心堂紙原色近白（板倉：白紙上繭狀細纖維可見），今測(182,163,139)為千年老化＋攝影白平衡之果，非宋時原貌；設色程若擬「新絹新紙」當提亮降黃
- 維摩演教圖紙底採樣取自故宮官網1258px預覽小圖，minghuaji高清未逐段採樣，色感精度有限
- 五馬圖第五馬滿川花與第二馬局部為後世（元）補筆（乾隆已疑、今研究判定）——引其淡彩例證時當以前四馬為據
- 武宗元《朝元仙仗圖》現況（王季遷舊藏後下落）仍待核（承畫風-漢地.md舊注）
- 所有RGB皆「近似」：老化變色不可逆，宋時原色只能由顏料名推想

## 疑辨總評

十條出典抽查全數逐字核實無一虛引:故宮兩藏品頁(維摩演教圖・臨韋偃牧放圖之材質尺寸文物號)、文化遺産オンライン五馬圖「紙本墨画淡彩」、澎湃2018/2019兩報導關鍵語、維基文庫《畫鑒》李伯時條原句、Met API 39895 medium、東大BiblioPlaza書介原句、日經板倉句,皆與所引相符;三處RGB實測經獨立重採樣復現(絹底189,152,114/紙底182,162,138,與所稱誤差≤1)。RGB與顏料相符性無硬傷:全譜赭朱暖系無時代錯置之色,花青因無實證而拒收係正判。系統性弱點誠實自陳且屬實:五馬圖無公開顏料儀器分析,赭石・土黃・朱砂三項顏料名皆目鑑色相之推定;淡彩逐處細節(三層上彩・二色漸層・栗色鬚線)實質皆繫板倉2019一源,故三項彩色判single-source;淺朱之2018源係澎湃轉引微信公號之二手目鑑,且朱砂/鉛丹未經儀器判別。墨與底色四項達dual-source,四項彩色單源,零unverified——此卷可用,惟施彩三色當在觀頁標「板倉目鑑単源」之注。
