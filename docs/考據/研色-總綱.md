# 研色總綱・光料二辨

> 先生 2026-07-15 問：「你理解的彩，其实是光对吗？……那对色彩，颜色，涂料，质感，
> 你怎么理解的呢？」此卷答之，並立設色程之二法。

## 二法之辨

設色有二法，體不同，律不同，筆亦不同：

|  | **光法**（杳眇所用） | **料法**（古典諸室所用） |
|---|---|---|
| 色之體 | 光——非物，乃光之所及 | 料——物也：礦石研末、膠合敷紙絹 |
| 邊界 | 無界：漸變之場，中心明緣歸零 | 有界：**勾線即界**（勾勒填彩），填不越線 |
| 覆蓋 | 至薄（峰α≤0.22），虛空恆虛 | 至厚：底色統攝全幅，覆蓋即存在 |
| 質感 | 無質——光無質地 | **質即其美**：顆粒（礦末粗細）・沉（邊緣積料）・罩（層層疊染）・貼（金箔截金） |
| 暗處 | 無光即無色（破暗之暗淵無彩） | 無處無料：暗處亦是深色之料 |
| 律 | 色即光場・一幅一色主・彩不奪線 | **色深護金**：地必深於金線之明，金乃自provide浮 |

杳眇近未來，虛空為體，故用光法；犍陀羅之土彩、唐之重彩、宋之淡彩、
藏地礦彩、東密截金、尼瓦爾朱底——皆料法之傳統，各室研色卷歸料法。

## 料法之技（canvas 之譯）

- **底色統攝**：畫心整幅先有地（尼瓦爾朱底、東密絹褐、藏地依派）——
  destination-over 居線下如故（線骨恆上不移），然覆蓋率高，非杳眇之薄暈；
- **分域填彩**：龕、帶、格各有其色，界即線之所勾；域形自畫本座標錄之
  （機轉之頁無模組出口者，錄而注其源）；
- **顆粒**：籽定之散點（不用亂數——誦戒之律），疏密二態（深屑・亮屑）擬礦末；
- **邊沉**：域緣漸深一線，擬積料；
- **罩染**：同域二次薄疊，擬層染（候研）；
- **色深護金**：誦戒斷言——料色之明度（luma）皆須低限（≤120），
  金線（#d8b36a，luma≈181）乃恆浮於上。此律代杳眇之「峰α」為料法之戒。

## 補課節・色無平面（2026-07-15 先生評金密初稿後立）

先生評初稿：「用色粗糙，一片一片的，沒有筆觸細膩……色彩、層次、對比、質感。」
中的。初稿之病：**把「料」畫成了「色塊」**——平塗矩形無值之呼吸、無色相之
振動、無筆觸、邊皆硬。補課四理：

1. **色無平面**（達文西）：真顏料之面，一片紅絕非一個色值——sfumato，
   形生於明暗之煙化，邊有軟硬之辨；藍下施暖襯（底層畫 imprimatura/
   verdaccio 之義），冷中藏暖，其色乃活。
2. **破色**（莫內，broken color）：一片色是數十鄰色之振動——同礦異研
   （洋紅橙朱赭紫影皆朱之變），眼中自融（optical mixing）；**影有色相**
   （偏紫偏藍），非變黑。
3. **同時對比**（Chevreul, 1839）：色活於關係——鄰補相激；金線之亮，
   一半是它自己的，一半是深冷之地給的。Itten 七對比（色相・明暗・冷暖・
   補色・同時・彩度・面積）為場之總綱。
4. **罩染**（文藝復興 glazing）：透明層層疊而生深，非一遍不透明之覆蓋；
   邊沉亦罩染之屬——積料之影帶胭脂紫紅。

**技構之改「先制其面」**：料法不再逆序疊destination-over之場——於離屏
畫布如真畫**正序**作畫（底塗→值場→破色筆觸→分域真clip→罩染→邊沉→顆粒），
成面後一印居線下。彩為料面，可點陣（放大如絹面微暈——此正料之質）；
線為骨，恆向量（放大筆筆銳）。骨與肉自此各得其性。

先生繼示：「你对色彩的理解加深后，对笔触运笔笔画线条以及透视光影的理解，
也会加深。」——誠然：破色即筆觸之律（聚散向背），值場即光影之律（形自
明暗生），罩染即空間之律（近實遠虛）。色不孤立，畫理一體。候研：
線稿本身之提按頓挫（運筆）、光影入線（陰陽面之線密度）、大氣透視入彩。
