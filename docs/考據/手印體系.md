# 考據 · 密教手印體系（為逐指造像用）

> Opus 考據代理報告（2026-07-05）；〔待核〕〔勿用〕如標。手印庫（bujian/shouyin，候鑄）之經據。
> 總判：①十指配當——左定右慧；五大配指小指＝地→大指＝空（通說可用，逐指定本卷次
> 〔待核〕，用須注「印相傳承說」）；②繪畫最緊者《大日經疏》卷十三（T39n1796）之
> **六種拳**；③Saunders《Mudrā》(1960) 可作主參框架，逐指幾何仍以疏文與四大圖像集校。

## 十指配當

- 左手＝定・理・止・月・福（蓮華部）；右手＝慧・智・觀・日（金剛部）。
- 五大配指：小指地・無名水・中指火・頭指風・大指空（合五輪五字之序）。
- 十度指名（對指成偶）：**檀慧**（兩小）・**戒方**（兩無名）・**忍願**（兩中）・
  **進力**（兩頭）・**禪智**（兩大）——印相書之指名體系。

## 印母

- **十二合掌**（大日經疏卷十三）：堅實心・虛心・未敷蓮・初割蓮・顯露・持水・
  **歸命（＝金剛合掌，十指交叉右壓左）**・反叉・反背互相著・橫拄指・覆手向下・覆手。

### 十二合掌逐指（疏文原典）

錄自《大日經疏》（大毘盧遮那成佛經疏）卷十三，T39n1796，CBETA（底本大正藏第 39
冊，頁 0714c09–0715a05）。疏引曰：「今此中先說十二種合掌名相。凡諸印法，此十二極
要，宜明記也。」下逐字錄疏文（〔　〕內為疏之夾註、音注：〔上〕上聲、〔入〕入聲、
〔二合〕梵二合、〔收間反〕反切）；「疏文原文」欄一字不改，「白話逐指法」欄乃畫師
可據以落筆之今譯。指名依疏自注：小指＝地指、無名指＝水指、中指＝火指、頭指（食指）
＝風指、大指＝空指。〔紋〕字從 CBETA 本，大正別本或作「絞」。

| 名 | 梵名音寫 | 疏文原文 | 白話逐指法 |
|---|---|---|---|
| 一・堅實心 | 寧尾拏（niviḍa） | 第一，合掌令當中心堅相著，十指頭稍相離少許開之，此名寧〔上〕尾拏〔上〕合掌〔此名堅實心合掌也〕。 | 兩掌正中相貼堅實，十指指尖略略分開、留少許縫。 |
| 二・虛心 | 三補吒（sampuṭa） | 第二，次以十指爪相當齊等，以指頭相合，掌中心小不相著，名三補吒合掌〔此名虛心合掌〕。 | 十指指甲對齊等高、指尖相合，唯掌心微不相貼、中留一空。 |
| 三・未敷蓮（未開蓮） | 屈滿囉（kuḍmala） | 第三，又次以十指頭相合，指亦齊等，然掌內空令稍穹隆，名屈滿囉合掌〔此云如未開蓮也〕。 | 十指尖相合齊等，掌心撐空、微微穹隆如未綻之蓮蕾。 |
| 四・初割蓮（初剖蓮） | 僕拏〔梵擬 bhagna？待核〕 | 第四，次以二地指二空指相著，餘指稍令開敷，名僕拏〔收間反〕合掌〔此名初剖之蓮也〕。 | 兩小指（地）相著、兩大指（空）相著，餘六指（食中無名）略張如初綻之蓮。 |
| 五・顯露 | 嗢多那若〔uttānaja；「那若」CBETA 有校勘，待核〕 | 第五，次又以兩掌仰而相並，令俱向上正相並鋪之，名嗢多那〔上〕若合掌〔此云露也〕。 | 兩掌仰起、掌心朝上，平正並排鋪陳。〔疏文止於掌勢，無逐指之詳〕 |
| 六・持水 | 阿陀囉（ādhāra） | 第六，次又並仰兩掌與前相似，而令諸指相就，稍令屈合之，如人掬水之狀〔勿令大屈也〕，此名阿陀〔上〕囉合掌〔此云持水也〕。 | 兩掌仰並如前，諸指相靠攏、微屈合，如人以手掬水之狀（勿過屈）。 |
| 七・歸命 | 鉢囉〔二合〕拏摩（praṇāma） | 第七，次又令十指頭相叉，皆以右手指加於左手指上，如金剛合掌也〔此云歸命合掌〕，梵音名鉢囉〔二合〕拏〔上〕摩合掌。 | 十指指尖交叉，一律右手指壓於左手指之上（即金剛合掌）。 |
| 八・反叉 | 微鉢哩哆（viparīta） | 第八，次又以右手加左反掌，以十指頭相紋，亦以右手指加於左手指上，名微鉢哩哆合掌〔此云反叉合掌也〕。 | 右手覆加左手成反掌，十指指尖相絞，仍右手指壓左手指上。 |
| 九・反背互相著 | 毘鉢囉哩曳薩哆（viparyasta） | 第九，次又以右手仰左手上，以左手覆在右手下，稍似坐禪人手相加之形，此名毘鉢囉哩曳薩哆合掌〔此云反背互相着合掌〕。 | 右手仰置左手之上、左手覆於右手之下，如坐禪者兩手相疊之形。〔疏文止於手相疊之勢，無逐指之詳〕 |
| 十・橫拄指（橫柱指） | 帝哩曳（tiryak） | 第十，次仰二手掌，令二手中指頭相接而仰之，名帝〔上〕哩曳〔入〕合掌〔此名橫柱指合掌〕。 | 兩手掌仰起，令左右兩中指指尖相接而仰之。 |
| 十一・覆手向下 | 阿馱囉（adhara） | 第十一，次又俱覆二掌，亦以二手中指相接，名阿馱囉合掌〔此云覆手向下合掌〕。 | 兩掌俱覆（掌心向下），亦令左右兩中指相接。 |
| 十二・覆手 | 同第十一（adhara）；疏不別立梵名（「亦同名也」） | 第十二，次又雙覆兩手，以二大指並而相接，十指頭向外，亦同名也〔亦云覆手合掌也〕。 | 兩手俱覆，兩大指並而相接，十指指尖朝外。 |
- **六種拳**：①金剛拳（大指入掌餘四指握）②蓮華拳/胎拳（四指握大指壓頭指外側節；
  大指立/壓兩式〔待核定本〕）③內縛（十指交於掌內）④外縛（十指交於掌外指頭露外）
  ⑤忿怒拳（金剛拳豎頭指小指）⑥如來拳（右金剛拳上、左蓮華拳下豎左大指入右拳）。
  「四種拳」為後世取捨，疏文原數六。

## 常用印逐指法（繪畫用）

| 印 | 逐指 |
|---|---|
| 智拳印 | 左金剛拳獨豎頭指；右金剛拳握之，右大指端拄左頭指端（「右小指握」之坊說為訛勿從） |
| 法界定印 | 兩手仰疊右上左下，兩大指端相拄成橢圓 |
| 彌陀定印/九品 | 基形同定印；品位＝大指捻頭指（上）/中指（中）/無名指（下）×手勢三分。**成立晚**（江戶《佛像圖彙》1690 系統化）——可繪勿托古，出典欄必注 |
| 觸地印 | 右手覆膝五指垂而指端觸地，左手仰臍前 |
| 與願印 | 掌向外五指舒展下垂 |
| 施無畏印 | 屈臂上舉掌向外五指舒伸 |
| 說法印 | 兩手當胸，大指與頭指（或中指）相捻成環，左內右外相對 |
| 金剛合掌 | 十指頭交叉右壓左（歸命合掌） |
| 期剋印 | 金剛拳豎頭指（或頭指小指如二鈎），威嚇鈎召（tarjanī） |

## 指節之度

拇指二節、餘四指三節（解剖定數）；T1419 掌7指5中指5而**不給三節內比**——
逐指幾何屬繪畫層自運，勿反推入量度層。

## 層次判定（寧缺毋誤）

- 可作出典：左定右慧、十二合掌、六種拳（疏卷十三）、十度指名、五字五輪。
- 通說可用注傳承：五指逐一配五大。
- 晚出可用勿托古：九品九式。
- **勿用**：手指配十二經絡臟腑（中醫嫁接）、現代瑜伽「指＝元素療癒」——非佛典，永不入庫。
