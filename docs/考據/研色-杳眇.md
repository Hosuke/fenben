# 研色卷・杳眇現代派

> 設色程開篇之卷。先生 2026-07-14 開示：「不同朝代，不同画风，用的色系和感觉，
> 颜料的质感，也是不同的。」杳眇為自派，色不繫古典出典，然必繫**派訣**與**材理**；
> 歷史諸室之研色卷（犍陀羅土彩・唐重彩・宋淡彩・藏地礦彩・東密截金・尼瓦爾朱底）
> 俟逐室考據，各立一卷，色譜≥2源乃錄（寧缺毋誤同律）。此卷唯論杳眇。

## 派色三律（承派訣五條而生）

1. **色即光場**：杳眇之色非顏料之敷，乃光之所及——一切色皆放射漸變或線性漸變之
   「場」，中心明而緣歸零，無一處平塗，無一處以輪廓為界。承派訣「金線自光」：
   線既自光，色便是光之暈，暈必有源（燈・珠・日月・佛身），無源之色不得落。
2. **一幅一色主**：承「一幅一勢」——一幅唯一主色相，餘色皆為應和之微光。
   《琉璃地》主琉璃青；《星迴》若設色主虛空玄青；《破暗》主珠暖而暗淵永無彩。
3. **彩不奪線**：金線恆為第一義。技構上彩層以 `destination-over` 居線下
   （線骨恆上為結構所保證，非自律所繫）；色律上單場之峰值透明度 **α ≤ 0.22**，
   誦戒斷言之。虛空恆虛：底色 #0d1124 之「無」即畫面之角色，色場不得漫溢全幅。

## 質感之議（礦彩之喻）

杳眇雖近未來，其色感取**礦彩薄罩**之理：如絹本背彩（裏彩色）自絹背透出，
色在線下、光自內生。漸變即暈染，疊場即積罩——`destination-over` 逐場後置，
恰似層層背敷。顆粒與絹紋俟後研（或以細噪點場為之，然噪必有律，不用亂數）。

## 候研之題

- 歷史諸室研色卷（每室色譜繫顏料名與現圖出典，≥2源）——調研隊已遣
  （yanse-liushi：六室研色＋疑辨核驗），回卷後親核乃錄；
- 顆粒・絹紋・截金質感之法；印製之彩校（深底彩印偏色，候實印回樣再調）。
