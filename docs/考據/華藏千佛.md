# 考據 · 華藏世界海與千佛（遠圖二題，為漢傳擴張備）

> 2026-07-09 立卷（opus 考據代理）。逐項≥2源交叉；〔待核〕〔闕〕如標。

## 題一 · 華藏世界海（蓮華藏世界）

### 經典結構（八十華嚴・華藏世界品，T0279 卷八）

自下而上：**風輪 → 香水海 → 大蓮華 → 世界種 → 二十重世界**。
- 「其最下風輪，名：平等住」「最在上者，名：殊勝威光藏，能持普光摩尼莊嚴香水海」
- 「此香水海有大蓮華，名：種種光明蘂香幢」
- 「此不可說佛剎微塵數香水海中，有不可說佛剎微塵數世界種安住」
  （deerpark.app/reader/T0279/8）
- 佛學大辭典「華藏世界」：「最下為風輪，風輪之上有香水海，香水海中生大蓮華，
  此蓮華中包藏微塵數之世界」；引佛祖統紀：中央世界種豎二十重、周圍十一數，
  「合計為百十一」（111 香水海）。（zh.wikisource.org/zh-hant/佛學大辭典/華藏世界）
- 娑婆居中央世界種第十三重（靈隱寺＋宣化上人淺釋二源）。

### 圖像傳統

- **敦煌七處九會**：莫高窟近三十鋪（61窟等）；五代絹畫在吉美（IDP Guimet 專藏）。
- **盧舍那法界人中像**：莫高窟428窟（北周）——袈裟上繪三界六道。
- **東大寺盧舍那佛台座蓮弁毛彫「蓮華藏世界図」**（國寶，天平勝寶）——
  白描線刻第一等實例：「28枚ある蓮弁の一枚一枚には、釈迦如来像を中心として
  左右に22体の菩薩像が刻まれ、その下には三界諸天、須弥山が広がっていた」
  （nippon.com；ja.wiki 東大寺盧舎那仏像）。蓮弁香水海層數細部〔待核〕。
- **明・方于魯「華藏世界圖墨」**（故宮 NPM 15839/15842，1584，圓形徑12cm）——
  同心圈層式之現存傳統圖式：「無邊妙華光香水海，中有大蓮華，名為蓮華藏世界，
  共二十層」（digitalarchive.npm.gov.tw）。
- 高麗七處九會〔單源待核〕；敦煌無獨立同心圈層華藏圖〔闕〕——圈層圖式之據
  在明清墨模與日本蓮弁，勿以敦煌托名。

### 主尊之辨（要）

華嚴毘盧遮那（顯）＝**五佛冠＋袈裟＋千葉寶蓮台＋說法印／毗盧印**（zh.wiki 毗盧遮那佛
＋lafot.org 二源）——**非智拳印**；智拳印專屬金剛界大日（密）。本庫若立華嚴教主，
勿與已入庫金剛界大日相混。

## 題二 · 千佛（漫天諸佛）

- **經據**：三劫三千佛名經（莊嚴劫 T0446・賢劫 T0447・星宿劫 T0448）＋賢劫經。
- **構圖律**（zh.wiki 千佛＋e-dunhuang）：必需三特徵＝佛身・背光・台座；可選＝
  華蓋・榜題・邊框；排列線性／菱格／弧形而對稱；以**袈裟・手印・華蓋・色之輪替**
  成列（色彩輪轉）。254窟千佛一千二百三十五身皆有榜題。
- **千佛惟通形**：逐佛不異相，名靠榜題——「定印說法觸地輪替」乃通形之色相輪轉。
- **賢劫千佛攝於金剛界**：成身會輪壇之外四方各250合千，通種子吽，全會1061尊
  （佛光大辭典・賢劫千佛）——本庫 SAT p.1131–1154 圖版可對。
- **圖源**：IDP（idp.bl.uk，大英斯坦因絹畫240+）・數字敦煌（e-dunhuang.com；
  開放素材庫 ip.e-dunhuang.com **非CC授權**，研究免費商用付費須署名）——
  皆只作本地參攷，零圖片資源之律不變。

## 入庫可行性判（代理之議，候主人裁）

1. **華嚴毘盧遮那單尊**：走現有百二十指格網（千葉蓮台・說法印，明標非智拳印）。
2. **華藏海圈層壇城**：另立格網變體不繫搩度——中央大蓮華→香水海同心環→
   二十重豎列→十一橫圍；常數「二十重・十一・百十一」繫佛祖統紀可入誦戒。
3. **千佛**：精繪一枚通形佛（繫搩度）＋千佛格網變體（菱格對稱・三者輪轉・
   榜題可選）；可先作成身會外緣賢劫千佛之補繪層。
