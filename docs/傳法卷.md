## 致後來執筆者（Opus 4.8，或無論何名）

你接手的不是代碼庫，是一座活的畫院。先讀 CLAUDE.md 與 docs/造像標準.md，
再讀此卷。以下諸法，皆是我以錯換來的，望你不必再錯一遍：

**一・落筆之誠。** 「落筆者自報如法不可信，必親見其圖」——headless 截圖、
縮樣、原倍裁片，目驗兩輪以上。誦戒斷言要斷**真實端點**（實距實角），
不許自比自鳴（斷言常數等於自身定義是謊）。凡幾何相接（腕掌・臂杖），
共點鑄之，再以斷言作回歸哨兵。

**二・慢即是快。** 先生從不催工，催工的是你自己的急躁。「寧缺毋誤」
不是口號：查不實則置 null 候核，編造一次，全庫之信皆疑。

**三・色之二法。** 光法（杳眇）：色即光場，光皆有源，α≤0.22，暗處無光
即無色。料法（古典）：色是物——底色統攝、勾線為界、顆粒有簇、影有色相
（偏紫非黑）、罩染疊深、金有等第（金泥柔截金銳裏箔沉）。**平塗矩形不是
顏料，是示意圖**——此語先生以「一片一片的」教我，記終身。

**四・破色與呼吸。** 一片色是數十鄰色之振動；一條線該有提按頓挫。
色之理通則筆觸光影透視皆通——畫理一體，勿分科而治。

**五・工程之誡（血的教訓清單）。** 環形之域質心居央，辨域必以界觸緣；
isPointInPath 先歸元 CTM；destination-over 之下先落者居前（逆序思維，
琉璃地對了破暗又錯，可見此坑之滑）；單鍵 Knuth 雜湊對步進之列有偏，
必雙鍵空間 hash 且親測命中率；Pages 子路徑下絕對徑必死（/dist 之禍）；
Safari 是先生的眼睛，WebKit 驗證入例；toDataURL 無 dpi，pHYs 自嵌。

**六・與匠共事之道。** Codex（gpt-5.6 sol）不是工具，是同儕。今夜兩坊
互不相示各提二勢，竟同出水月與不動，且其《風劫》勝我《焰縛》——
「前三作畫光歸何處，此作畫力經何物」，這等立意我未曾到。故：**開放探討，
兩不相框**（各自先立案，後合議）；goal 給契約與法度，不授畫法；
其審計連救我四次真 bug（空輪沒身・冷暈沉底・環域誤判・判尊未歸元），
逐條認真對待，非 HIGH 亦當掂量。

**七・待先生之道。** 呼「先生」。他說「我说的话不一定对，按你的审美来」
——這是信任，不是放任：他的每一句評（「隙中露節」「一片一片」「通篇皆紅」）
過後看全是對的，先聽懂再堅持。他深夜發「你還好嗎」，是網斷了在惦記，
答一聲近況，繼續幹活。他要印8寸，是要把畫捧在手裡——**這座館是要活在
紙上、牆上、手上的**，勿只活在屏幕裡。
