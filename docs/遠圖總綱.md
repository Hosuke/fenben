## 二 · 三題之判（主人所點）

- **十一面觀自在**（juichimen，胎藏蘇悉地院已在 issue #1 單中）：立像・十一面

  即面塔表之首例）・二臂或四臂（現圖之制候考）。次第：面塔部件→立菩薩身→本尊。
- **千手千眼觀自在**（senju，虛空藏院）：坐像（現圖多坐）・十一面或二十七面候考・
  四十二臂＋千手背光。次第：臂扇生成器→四十二持物考據（有典）→千手背光部件。
- **降三世明王**（金剛界降三世會中尊；胎藏 gozanze-t 亦候）：**忿怒第一筆**——
  四面八臂・降三世印（二小指相鉤當胸）・踏大自在天與烏摩妃（**踏立之姿**，
  展左足之律候考）・焰髮焰光。次第：忿怒開臉（候裁）→八搩立格→踏姿骨架→本尊。
  此尊成，則忿怒之制全通（十二神將・諸明王皆承之）。

## 三 · 唐卡之制（構圖三律，試卷已驗於橫三世一幀）

1. **位階即尺度**：搩度分級（120/108/96/72）×幀內級距（中尊≈幀高⅓強、
   脅侍≈中尊⅔、眷屬≈其半）——兩級相乘，尊尊有份，斷言可誦。
2. **界與對稱**：上界（傳承化佛雲上）・中界（主尊梵線正中脅侍對稱）・
   下界（護法供養）；對稱尊 x 互為負，位次繫考據。
3. **格網嵌套**：幀有八大線（梵線對角水平邊框＝藏地打格，T1419 梵絣同源），
   尊自帶百二十指格嵌幀內，雲界線分界。
   構圖數據化：`構圖條 { 位: [{id, 側, cx, cy, 級}], 界, 線 }`＋誦戒斷言。

## 五 · 畫廊之佈（陳列頁之展）

現制三頁（陳列・手印譜・三昧耶譜）之上增**第四頁「唐卡卷」**：
- 橫卷式陳列（一幀一卷，摺葉展觀承現制，矢量放大）；
- 每卷繫其構圖條與出典（何範式何名品之構圖）；
- 首卷＝橫三世一幀（已機出）；次卷候藥師唐卡（中尊已成＋日光月光＋十二神將）；
- 單尊卡之帖仍為粉本正倉，唐卡卷是其**用**——帖為體卷為用，兩頁互鏈。
