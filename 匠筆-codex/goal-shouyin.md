# Goal · 手印廿七式（Codex 獨立自運）

你是繪佛手印的畫師兼工程師。在 `匠筆-codex/` 下用 **canvas 白描**畫出金剛界手印
廿七式，成一頁 `匠筆-codex/shouyin-codex.html`（直開 `python3 -m http.server` 於倉根
即可看）。畫法完全由你自創——不要參照 `src/bujian/` 里任何現有畫法（這是兩坊比較
實驗），但**儀軌內容必須依典**。

## 依典（必讀）

`docs/考據/手印體系.md` ——逐指法皆在此卷。要畫的廿七式：

- **十二合掌**（大日經疏卷十三）：堅實心・虛心・未敷蓮・初割蓮・顯露・持水・
  歸命（＝金剛合掌，十指頭交叉右壓左）・反叉・反背互相著・橫拄指・覆手向下・覆手
- **六種拳**：金剛拳（大指入掌餘四指握）・蓮華拳（四指握大指壓頭指外側節）・
  內縛（十指交掌內）・外縛（十指交掌外指頭露外）・忿怒拳（金剛拳豎頭指小指）・
  如來拳（右金剛拳上左蓮華拳下豎左大指入右拳）
- **常用九印**：智拳印（左金剛拳獨豎頭指右拳握之，右大指端拄左頭指端——
  「右小指握」坊說為訛勿從）・法界定印（右上左下二大指端相拄）・彌陀定印
  （大指捻頭指，九品晚出須注）・觸地印・與願印・施無畏印・說法印（拇頭相捻成環
  左內右外）・金剛合掌・期剋印（金剛拳豎頭指如鈎）

每卡題註：印名＋一行逐指法（錄自考據卷）＋出典；晚出/坊說/待核以朱色注。

## 條件

- **筆具可用可不用**：`dist/bi.js` 有 `執筆(ctx, R)` 之筆（`u`＝一指＝R×0.0145、
  `Y(z)` 縱錨、`運筆()` 三等筆骨/衣/細/逸帶筆鋒起收）。用它或裸 canvas 自便。
- **手之量度**（T1419）：掌廣五指、中指長五指、全手約十二指；指三節、拇二節。
- **質之判**：逐指逐節可辨；**畫其可見**——線遇前物必斷（遮擋要真，如
  參攷-local/五部心観/SAT図像部 之唐代白描，可開圖細看其遮斷之法）；圓滿之相
  （弧不露稜）；線有粗細層次。
- **頁式**：紺紙金泥——bg #0d1124、金泥 #d8b36a、金明 #f2d9a4、絹 #e8e1cf、
  朱 #b0402c、墨淡 #7a7f9a，襯線字（Noto Serif TC 一族）。卡式網格陳列，
  canvas 每卡 ≥300px。分三節（十二合掌／六種拳／常用九印）。

## 自驗（必行）

headless 截圖自校**兩輪以上**：playwright-core（~/.npm/_npx 快取內有）＋
chrome-headless-shell（~/Library/Caches/ms-playwright；直 --screenshot 會掛 rAF，
必以 playwright 定時截）——照抄 `tools/圖錄.cjs` 之 覓playwright/覓headless 即可。
逐卡看圖：指數對不對、遮擋斷沒斷、印相合不合逐指法。所見所改記入
`匠筆-codex/shouyin-筆記.md`。
