# 粉本 · fenben

> 「毗尼藏者，是佛法壽命；毗尼藏住，佛法亦住。」——《善見律毗婆沙》
>
> 雲施法雨，留形住世。

`fenben`（粉本）者，如法造像之標準庫也。歷代畫師刺孔扑粉、形隨本轉，其底稿謂之
粉本（實物之錨：大英博物館藏敦煌五尊佛刺孔粉本，編號 1919,0101,0.72，對折同刺
以保對稱，扑粉之痕猶存）。此庫即信息時代之粉本：量度為數據，儀軌為記錄，筆法為
函數，校驗為斷言——形經千手轉寫而不失真。

## 旨

律者，法之壽命；約束在，則法住。造像亦然：《佛說造像量度經》通篇是數——幾指幾麥、
何處至何處幾分——所規定者非某一張畫，乃**約束系統**。畫師硃線打格，是人肉執行此
參數；此庫以代碼執行之，如法遂成可斷言（assert）之性質。

- **量度為骨**——格網數值逐字出經（大正藏 T21n1419），頁碼錨於行間。
- **依軌為形**——逐尊規範記錄，字段皆繫出典；信未至「已核」者不得上壇。
- **寧缺毋誤**——出典查不實，退種字／三昧耶形，不編造錯相。
- **載入即誦戒**——量度和數斷言，破則擲；每次提交重跑校驗，即半月半月誦戒。

git 之結集，CI 之布薩，fork 之傳戒。迦葉留形雞足山以待彌勒，鑑真傳律而乾漆像
住世唐招提寺——傳律與留形，一事兩面：**律在則形正，形存則法傳。**

## 法

庫分三層，皆 ESM，標識符逕用漢字（JS 本許 Unicode 標識符，非誤植也）：

1. `fenben/liangdu.js`（**量度**）——單位鏈、縱剖橫剖、坐像折算、尊格搩度階梯、
   錨點生成、誦戒斷言。
2. `fenben/yigui.js`（**儀軌**）——逐尊之尊・形・搩度・面臂・印・持物・身色
   （経軌／現図兩軌並錄）・座・光，字段皆帶出典引文與異說。
3. `fenben/baimiao.js`（**白描**）——格網驅動之筆法：格網定其骨，印法定其手，
   形類定其衣。畫者不自作主張。

一切尊形程序而生，**零圖片資源**；諸源只作考據引用，圖像絕不入庫——借相正相，
非以相代相。

## 用

```js
import { 錨點, 搩度, 誦戒 } from './fenben/liangdu.js';
import { 儀軌, 已核之 } from './fenben/yigui.js';
import { 白描 } from './fenben/baimiao.js';

const 面 = 已核之('center', 'k');   // 金剛界大日，信＝已核乃得，否則 null
if (面) 白描(ctx, 半徑, 面);         // 依格網落筆：菩薩形，智拳印
```

試筆：倉根起 `python3 -m http.server`，開 `shibi.html`——諸尊白描並陳，候主人過目。

## 緣起

析出自 [mandala](https://github.com/Hosuke/mandala)（金胎不二曼荼羅引擎，
2026-07-03）：五佛淬火既成，標準乃出為獨立之庫。mandala 內之孿生渲染層曰
`funpon.js`——同字兩讀，漢音 fenben，和音 funpon，金胎漢和兩土之音也。
所志者遠：漢傳漫天諸佛（三世佛、五方佛、八大菩薩、羅漢、諸天、水陸神譜）與
藏地量度二宗（勝樂百二十、時輪百二十五，並陳不強合），皆候逐波考據入庫。
