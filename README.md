# 粉本 · fenben

> 「毗尼藏者，是佛法壽命；毗尼藏住，佛法亦住。」——《善見律毗婆沙》
>
> 雲施法雨，留形住世。

`fenben`（粉本）者，如法造像之標準庫也。歷代畫師刺孔扑粉、形隨本轉，其底稿謂之
粉本（實物之錨：大英博物館藏敦煌五尊佛刺孔粉本，編號 1919,0101,0.72，對折同刺
以保對稱，扑粉之痕猶存）。此庫即信息時代之粉本：量度為數據，儀軌為記錄，筆法為
函數，校驗為類型與斷言——形經千手轉寫而不失真。

## 旨

律者，法之壽命；約束在，則法住。造像亦然：《佛說造像量度經》通篇是數——幾指幾麥、
何處至何處幾分——所規定者非某一張畫，乃**約束系統**。畫師硃線打格，是人肉執行此
參數；此庫以代碼執行之，如法遂成可斷言之性質。

- **量度為骨**——格網數值逐字出經（大正藏 T21n1419），頁碼錨於行間。
- **依軌為形**——逐尊規範記錄，字段皆繫出典；信未至「已核」者不得上壇。
- **類型即律**——TypeScript 閉集聯合鎖形類・印名・信級於編譯期，寫錯一字，編譯即破戒。
- **寧缺毋誤**——出典查不實，退空月輪，不編造錯相。
- **載入即誦戒**——量度和數斷言，破則擲；每次提交重跑校驗，即半月半月誦戒。

git 之結集，CI 之布薩，fork 之傳戒。迦葉留形雞足山以待彌勒，鑑真傳律而乾漆像
住世唐招提寺——傳律與留形，一事兩面：**律在則形正，形存則法傳。**

**精度之的：當代犍陀羅之工。** 非示意之圖標，乃專業繪佛之白描——骨相安於格網，
衣褶有起伏頓挫，手指逐節如儀，開臉依相好之目。

## 法

源在 `src/`（TypeScript），出在 `dist/`（ESM ＋ d.ts，`npm run build` 而得）：

1. `liangdu.ts`（**量度**）——單位鏈、縱剖橫剖、坐像折算、尊格搩度階梯、錨點、誦戒。
2. `yigui.ts`（**儀軌**）——逐尊之尊・形・搩度・面臂・印・持物・身色・座・光，
   字段皆帶出典引文與異說；`面` 之類型即其律。
3. `bi.ts`（**筆具**）——格網座標之共用筆法：折線・弧・曲線・鬚線・淡墨，
   單位皆自指，逐尊之筆與通形共執之。
4. `zun/`（**落筆簿**）——逐尊親筆之地：每尊一檔，於 `zun/index.ts` 登簿
   （鍵＝`'尊號|側'`）。
5. `baimiao.ts`（**白描總門**）——有專筆用專筆，無則退通形（形類之基準白描）。

一切尊形程序而生，**零圖片資源**；諸源只作考據引用，圖像絕不入庫——借相正相，
非以相代相。

## 用

```ts
import { 已核之 } from 'fenben/yigui';
import { 白描 } from 'fenben/baimiao';

const 面 = 已核之('center', 'k');        // 金剛界大日，信＝已核乃得，否則 null
if (面) 白描(ctx, 半徑, 面, 'center|k'); // 依格網落筆：菩薩形，智拳印
```

陳列：倉根 `npm run build` 後起 `python3 -m http.server`，開 `index.html`——
紺紙金泥之壇，三十七尊滿譜：虛月輪候筆，朱界候審，鈐印已核；點尊開帖，
五百指大圖與逐條出典並陳。
